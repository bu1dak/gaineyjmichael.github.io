The purpose of this website is to showcase my abilities with web languages by using HTML5 and CSS3. The template that I chose helped to show my abilities to edit an existing CSS and other existing HTML pages. This website will also showcase my hobby of photography. Coding a website is much like photography and cooking to me. It takes the right amount of time and patience to build a website or code a program. As with photography and cooking, the timing has to be right. While cooking is more by the book, photography gives you the ability to be creative because there really aren't any rules. My hobbies reflect my personality in that, while I can be creative, I also enjoy having that organization and patience that cooking requires.
